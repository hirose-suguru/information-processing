# VSCodeでMarkdownを良い感じに見よう
## VSCodeのインストール
VSCodeを入れましょう。VSCodeはMicrosoftが出しているIDE(Integrated Development Environment, 統合開発県境)で、めっちゃ使いやすいメモ帳をさらに魔改造させた感じのものです。
下のリンクからダウンロードできます。

https://code.visualstudio.com

でインストールを進めていこうとするとインストール先が
```
C:\Users\[ユーザー名]\AppData\Local\Programs\Microsoft VS Code
```
になってるかを確認してください。<br>
もし`C:\Program Files`に入れてこようとしてたら上のパスに自分のユーザー名を入れて書き換えて下さい。

オプションに関しては<br>
「追加タスクの選択」<br>
というので<br>
**「エクスプローラーのファイルコンテキスト～アクションを追加する」**<br>
**「エクスプローラーのディレクトリコンテキスト～アクションを追加する」**<br>
**にチェックを入れておくのを推奨します。**<br>
「デスクトップ上にアイコンを作成する」はお好みで。どうせタスクバーにピン留めして開くので僕は使ったことがありません。

## 拡張機能を入れよう。
VSCodeを開いたら左の方に、箱3つと傾いた箱のやつがあるのでクリックしてください。
すると、Extensionと書かれているやつがニョキっと出てきます。これがVSCodeにさらに拡張機能を入れられるやつです。<br>
そこの検索欄に **Japaneseって打って、Japanese Language Pack for Visual Studio Codeやつを入れて下さい。** Microsoftが出してるやつで、14Mとかってなってるやつです。<br>
すると「再起動するけどいい？」みたいなのが出るので「ええで～」みたいな感じのやつを選んで再起動してください。

同様に **Markdown Live Editorと打ってそれを入れてください。** 7Kとかってなってるやつで、Jun Ishiiさんという日本人の人が1か月くらい前に出したやつです。<br>
それをインストールしましょう。

## Markdown Live Editorを使って良い感じに見よう。

- VSCodeの左上の「ファイル」をクリックして「ファイルを開く」からダウンロードした`information_processing.md`を開いてください。

- そして、`information_processing.md`と書かれているタブを右クリックして「Open with Markdown Live Editor」をクリックすると、隣に同じファイルができるので、クリックしてそっちに移動してください。

- これでプレビュー状態になっており、見出しのところの小っちゃい▽マークで折りたたむことができるので良い感じに見て下さい。

- 暗い画面が嫌な方は左下の歯車マークから「テーマ」→「配色テーマ」でLight系のテーマを選んだりして自由に良い感じにやってください。

- また、左端のところの1番上のファイルみたいなやつを開くと「エクスプローラー」と出てきて、そこの下に「MARKDOWN OUTLINE」というのがあり、それで簡単に見出しをぴょんぴょんできます。
