# 3. おすすめのYouTubeチャンネル ― ゆるコンピューター科学ラジオ
多くの人にとってエンジニアリングの世界はよく分からないしつまらないものですが、そういう中で少しずつ知識をつけていくなら楽しく学べることが大事だと思います。<br>
ということで、僕の好きなYouTubeチャンネルである、ゆるコンピューター科学ラジオを紹介します。

https://www.youtube.com/@yurucom

このチャンネルは、慶応の理工学部卒の堀元見さんが、機械音痴の相方 水野大貴さんにコンピューターのことをあれこれ教えていくチャンネルです。真面目な感じではなく、30歳くらいの大人が2人でわちゃわちゃやっているので楽しく見れると思います。

個人的にはターミナルを使ってみる回や、プログラミングをしてみる回、他にはダイクストラ法というアルゴリズムを扱う回やVRの回、ブラウザの回が好きです。プログラミングの回は皆さんもプログラミングをするので楽しく見れると思います。<br>
URLを貼っておきます。

https://youtu.be/ijXMssBJvpg?si=36hXdmwmaGNsQvn2
